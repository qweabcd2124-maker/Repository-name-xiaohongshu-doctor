## 一、2 分钟精简版（提交用）

### 技术要求
- 分辨率：1920×1080 或更高
- 格式：MP4，H.264
- 录屏工具：OBS / QuickTime / 系统录屏
- 配音：可选（加字幕更稳妥）
- 背景音乐：轻柔不抢戏

### 录制注意事项

1. **提前清理浏览器** — 关闭无关标签页、通知、书签栏
2. **用一个好看的示例笔记** — 选一篇美食类的，截图清晰、标题有改进空间
3. **鼠标移动要慢** — 评委需要跟上你的操作
4. **如果诊断等待超过 30 秒** — 剪辑跳过等待时间
5. **字幕用白底黑字** — 清晰易读，不用花哨效果

## 二、决赛现场 Demo（2 分钟调试期间）

如果选择**现场 Live Demo**而非视频：

### 预置状态
- 浏览器已打开 xiaohongshu-doctor.muran.tech
- 已经有一张截图在剪贴板（Ctrl+V 直接粘贴）
- 标题预先想好但不预填（现场打字更有真实感）

### 备用方案
- 如果网络卡 → 立刻说"切到我们的预录版本"
- 如果 API 超时 → fallback 自动生效，继续讲

### 加分动作
- 让评委说一个小红书笔记的主题 → 现场即兴诊断
- "您最近发过什么小红书吗？截图给我们试试？"
